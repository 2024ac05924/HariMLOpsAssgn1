# Heart Disease Prediction using MLOps

## Project

Machine Learning Operations (MLOps) Assignment 1

## Objective

The objective of this project is to develop an end-to-end machine learning pipeline for predicting the presence of heart disease using the UCI Heart Disease dataset.

The project includes:

- Data acquisition
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Machine learning model development
- Experiment tracking using MLflow
- REST API using FastAPI
- Docker containerization
- CI/CD pipeline
- Kubernetes deployment
- Monitoring

## Dataset

Heart Disease UCI Dataset

## Models

- Logistic Regression
- Random Forest
- XGBoost
